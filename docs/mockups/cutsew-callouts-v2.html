What each card holds

A
Large main image — the hero close-up, as big as the card allows (about ⅔ of the image strip).
B
Smaller supporting image — a second reference (a different angle, a detail, a "do/don't"). Clearly visible but smaller than the main.
C
Auto-fill — if you don't add a supporting image, the main image grows to fill the whole strip (see call-out 4 above).
D
Numbered badge + title + description — the badge number is the same number as the dot you placed on the garment, so the factory can find it instantly.
How the dots work now: the left image is a plain photo of the whole garment. You click where a detail is and the app drops that call-out's red number there; click again (or drag) to move it. One dot per call-out — 1–4 on page 07, 5–8 on page 08.
