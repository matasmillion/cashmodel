MULTI-FABRIC · PAGINATION VIA DROPDOWN

What happens when you pick 2+ fabrics

Page 03 keeps the same chrome, but the top of the body adds Fabric N of M ▼ nav. Each fabric gets its own A4 page. The dropdown lets vendors jump between them; the PDF export emits one page per fabric back-to-back.
Fabric 1 of 2 ▼ · Body
Fabric 2 of 2 ▼ · Rib
Total tech pack length grows by 1 per extra fabric (single fabric = 24 pages, two fabrics = 25, etc.). The dropdown is interactive in the live preview and inert in the PDF export — vendors flip pages instead. Note: for body+rib hoodies you'd usually pick a knit fabric whose library card already has the matched ribbing (ZF-RIB-340-A above) — and that ribbing only triggers a separate page if you've intentionally added it as a 2nd BOM line.
