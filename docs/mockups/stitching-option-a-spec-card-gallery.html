What each card holds

A
Photo on top — the stitch close-up, same shape as a call-out's main image.
B
Numbered badge + name + code — the red badge is the call-out badge; the name is the stitch in plain words; the code (504, 406…) is the Stitch Type field, shown in monospace.
C
Spec mini-sheet — Operation, Seam type, Machine, SPI/SPCM, and Thread (colour chip + type). Every one of the eight table fields is here, just stacked under its own photo instead of in a faraway table.
D
Notes strip — the per-factory translatable note, on a Salt band at the foot of the card.
Nothing lost: all six photos, and all eight spec fields per stitch (Operation, Seam Type, Stitch Type, Machine, SPI/SPCM, Thread Color, Thread Type, Notes) live on the cards. The garment-block dropdown and the full Cut & Sew calculator at the bottom — labor cost, Estimate with AI, the AI estimate, the CMT note, and the Cost Chat — are unchanged.  ·  Built on 07/08: same white card, 0.5px Sand border, red numbered badge, FR type and colours.  ·  Keep in sync when we build: the Style editor (TechPackSteps.jsx), the live preview / PDF (TechPackPagePreview.jsx), the reusable library card (CutSewBuilder.jsx), and its schema (cutSewLibrary.js) all draw Stitching, so all four change together.
