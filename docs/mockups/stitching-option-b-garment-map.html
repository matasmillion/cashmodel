Spec pills — Stitch Type, Machine, SPI/SPCM, and Thread (colour chip + type), with Operation as the subtitle. All eight fields are present; the pills keep them tight so the photo and the map breathe.
D
Notes line — the per-factory translatable note at the foot of the card.
